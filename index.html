Carma Bootstrap

This is a simple HTML5 web application that showcases different parts of the Carma API to illustrate how an application can be built.

Everything here is fully functional, so that you can see the code involved for login and user signup. Discovery and other functionality will be added in the coming days

How to use this demo

If you have already got a developer account with Carma, you can use it to log in using the controls on the top right corner.

If you don't have an account yet, then you can create an account using the sign up page
